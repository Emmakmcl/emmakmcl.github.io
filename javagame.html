Multi-level Java Game

This is a multi-level 2D game made with the City Engine Library. It consists of 2 packages, 40 java classes. Interfaces from the City Engine Library such as Collision Listener, Action Listener, Step Listener were used for the implemention. Furthermore, I made use of inheritance with classes such as Mouse Adapter, User View, World, Static Body, Key Adapter (all are classes of the City Engine Library) for efficiency and encapsulation.
